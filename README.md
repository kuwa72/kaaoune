# kaoune (カオウネ)

家を買うためにインターネットで公開されている物件情報を比較するシステム。

## 機能要件
1. **データ取り込み**: ユーザーが物件ページのURLをコピペして登録。
2. **データ抽出**: ローカル環境でブラウザ自動操作（Playwright等）を行い、ページから情報を解析・取得。ボット対策を回避しつつ情報を収集。
3. **比較機能**: 価格、広さ、間取りに加え、**「月々の支払額」**を重要指標として比較。
4. **UI**: グラスモーフィズムなどを取り入れたモダンでリッチなWebインターフェース。

## スタック
- **Framework**: Next.js
- **Styling**: TailwindCSS, Framer Motion
- **Scraping**: Playwright
