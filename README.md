家を買うためにインターネットで公開されている物件情報を比較・管理するシステムです。
気になった物件のURLを貼り付けるだけで、情報を自動で取得し、一覧で見やすく整理できます。

## 主な機能

- **物件登録**: URLをコピペするだけで、各種不動産ポータルサイトから情報を自動取得。
- **比較**: 価格、広さ、間取りに加え、**「月々の支払額」** で比較可能。
- **管理**: 「検討中」「内見予定」などのステータスで物件を管理。

## はじめかた

エンジニア知識がなくても以下の手順で利用を開始できます（現在はローカル環境での利用が前提です）。

### 事前準備

1. **Node.jsのインストール**:
   [Node.js公式サイト](https://nodejs.org/)から「推奨版（LTS）」をダウンロードしてインストールしてください。

2. **ソースコードの取得**:
   このページ右上の「Code」ボタンから「Download ZIP」を選んでダウンロードし、解凍してください（またはGitを使用）。

### インストール

解凍したフォルダを開き、その場所でターミナル（PowerShellやコマンドプロンプト）を開いて以下のコマンドを入力します。

```bash
# 必要なソフト（ライブラリ）をインストールします
npm install
```

### 使い方

1. **アプリを起動する**:
   以下のコマンドを入力してアプリを立ち上げます。
   ```bash
   npm run dev
   ```
   
2. **ブラウザで開く**:
   起動したら、ブラウザで [http://localhost:3000](http://localhost:3000) にアクセスします。

3. **物件を追加する**:
   画面上部の入力欄に、物件詳細ページのURLを貼り付けて「追加」ボタンを押します。
   （少し時間がかかりますが、自動で情報を取得してカードが追加されます）

### 外部公開（オプション）

スマートフォンの実機で確認したい場合など、外部からアクセスできるようにするには以下のコマンドを使います。

```bash
# ngrokを使ってローカルサーバーを外部に公開します
npm run share
```

実行後、ターミナルに表示されるURL（例: `https://xxxx.ngrok-free.app`）にアクセスしてください。
※ 初回利用時はngrokのアカウント登録とトークン設定が必要になる場合があります。

## 画面・機能マニュアル

### 1. 画面構成
- **ヘッダー**: 
  - **ソート**: 「登録順」「支払額順」「広さ順」などで物件を並び替えられます。
  - **選外表示**: 「選外」ステータスの物件を表示/非表示切り替え。
  - **表示モード**: カードビュー（グリッド）とリストビューの切り替え。
  - **設定（歯車アイコン）**: ユーザー管理やローン設定を行います。
- **検索バー**: 物件URLを入力して「追加」する場所です。
- **物件リスト**: 登録された物件がカード形式で並びます。

### 2. 物件カードの見方
- **ステータス**: カード左上のラベル（例：検討中、内見予定）。クリックして変更可能。
- **価格・支払い目安**: 物件価格と、管理費・修繕費・ローンを含めた「月々の支払総額（目安）」を表示。
- **基本情報**: 
  - **広さ**: 専有面積（50m²未満は赤字警告）。
  - **駅**: 最寄り駅と徒歩分数。
  - **築年数**: 築年数と経過年数（旧耐震基準は赤字警告）。
  - **総戸数 / 権利**: 戸数と所有権/借地権の区別。
- **ユーザー評価**: 登録ユーザーごとの「いいね/うーん」評価とコメント。

### 3. 操作方法
- **物件情報の編集**: カード右上の「設定（歯車）」アイコンから、価格や広さなどの数字を手動で修正できます（自動取得が間違っていた場合など）。
- **情報の更新**: 「更新（矢印）」アイコンで最新の掲載情報を再取得します。
- **削除**: 「ゴミ箱」アイコンでリストから削除します。
- **比較モード**: カード左上のチェックボックスを選択すると、画面下部に操作バーが出現。「比較する」ボタンで、選択した物件を横並びで比較できます。

### 4. 設定（シミュレーション用）
ヘッダーの歯車アイコンから設定を開けます。
- **ユーザー設定**: 家族やパートナーを登録し、それぞれのアイコンで評価・コメントができます。
- **ローン・計算設定**: 「想定金利」「借入期間」「頭金」を設定すると、物件カードの「月々の支払額」が自動で再計算されます。

## 技術情報（エンジニア向け）
